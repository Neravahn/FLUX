# FLUX 

**FLUX** is an interactive Python web application for exploring stock prices, technical indicators, and market trends. It combines real-time financial data with interactive charts to provide a user-friendly and exploratory experience.

- **Real-time stock data:** Fetch historical and intraday stock prices via Yahoo Finance.
- **Technical indicators:** OBV, VWAP, CMF, ADL, moving averages.
- **Interactive charts:** Zoom, pan, toggle grid, and hover tooltips.
- **Customizable analysis:** Select tickers, benchmarks, intervals, and periods for indicators.
- **Download charts:** Export visualizations as PNG images.
- **Responsive UI:** Works on desktop and mobile devices.
